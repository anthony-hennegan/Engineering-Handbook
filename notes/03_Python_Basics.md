## Purpose

A Python program is a set of instructions written in the Python programming language that tells the computer what to do.

When the program is executed, Python reads the instructions from top to bottom and performs each operation in order unless directed otherwise.

## Key Concepts
### Statement
A statement is a single instruction that tells Python to perform an action.

### Variable
A variable is a named location that stores a value so it can be used later in the program.

### Value
A value is a piece of information stored or used by a program.

### Data Type
A data type is an attribute of a value that tells Python how to store, interpret, and operate on that value.
