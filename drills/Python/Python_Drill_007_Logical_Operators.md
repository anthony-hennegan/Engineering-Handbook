# Python Drill 007 - Logical Operators

## Objective

Practice using `and`, `or`, and `not` to combine or reverse conditions.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Create a variable named `is_open` and set it to `True`.
- [ ] Create a variable named `checkout_limit` and set it to `3`.
- [ ] Ask the user if they want to view the book selection.
- [ ] Clean the response using `.strip().lower()`.
- [ ] Accept both `yes` and `y` as valid yes responses.
- [ ] Accept both `no` and `n` as valid no responses.
- [ ] If the user says yes or y, ask how many books they want to check out.
- [ ] Convert the requested book count to an integer.
- [ ] Approve checkout only if the library is open and the requested number is less than or equal to the checkout limit.
- [ ] If checkout is not approved, print a message saying checkout is not available.
- [ ] Use `not` to print a message if the library is closed.
- [ ] Add comments explaining what `and`, `or`, and `not` do.
