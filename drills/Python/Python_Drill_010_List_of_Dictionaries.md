# Python Drill - Search a List of Dictionaries

## Objective

Practice searching through a list of dictionaries using a loop, a condition, and a found flag.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory
- Use your previous drill only after your first attempt

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Create a list named `books`.
- [ ] Add at least three dictionaries inside the `books` list.
- [ ] Each dictionary should represent one book.
- [ ] Each book dictionary should have:
  - `title`
  - `author`
  - `checked_out`
- [ ] Create a variable named `search_title`.
- [ ] Set `search_title` to `"Dune"`.
- [ ] Create a variable named `found_book`.
- [ ] Set `found_book` to `False`.
- [ ] Use a `for` loop to loop through the `books` list.
- [ ] Inside the loop, check whether the current book’s title matches `search_title`.
- [ ] If the title matches, print the book title and author.
- [ ] If the title matches, change `found_book` to `True`.
- [ ] After the loop, check whether `found_book` is still `False`.
- [ ] If `found_book` is still `False`, print `"Book not found."`
- [ ] Change `search_title` to a book that does not exist in the list.
- [ ] Run the program again and confirm `"Book not found."` prints.
- [ ] Add a comment explaining what `found_book` is used for.
- [ ] Add a comment explaining why the `"Book not found."` check happens after the loop.
- [ ] Add a comment explaining what `book["title"] == search_title` checks.
