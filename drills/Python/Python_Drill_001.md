# Python Drill 001

## Objective

Practice creating, using, and updating variables without referring to notes.

---

## Rules

- No handbook
- No flashcards
- No internet
- Work from memory

---

## Tasks

- [ ] Create a new Python file.
- [ ] Create at least four meaningful variables using appropriate names.
- [ ] Use at least three different data types.
- [ ] Print each variable.
- [ ] Reassign one variable to a new value.
- [ ] Print the updated value.
- [ ] Demonstrate that changing one variable does not affect another variable.
