# Python Drill 009 - Loops

## Objective

Practice using `for` loops to repeat actions for every item in a list.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Create a list named `books` with at least four book titles.
- [ ] Use a `for` loop to print each book.
- [ ] Print a heading before the loop.
- [ ] Print `"Done"` after the loop.
- [ ] Add a new book to the list using `.append()`.
- [ ] Use another `for` loop to print the updated list.
- [ ] Create a variable named `book_count` and set it to `0`.
- [ ] Use a loop to count how many books are in the list.
- [ ] Print the final `book_count`.
- [ ] Use a loop and condition to check whether `"Dune"` is in the list.
- [ ] Print `"Found Dune."` only when the current book is `"Dune"`.
- [ ] Add a comment explaining what the loop variable does.
- [ ] Add a comment explaining what happens when the condition inside a loop is False.
