# Functions

## Objective

Practice defining functions, calling functions, and passing values into functions.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory
- Use the exercise file only after your first attempt

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Define a function named `show_message`.
- [ ] Inside the function, print a simple message.
- [ ] Call `show_message`.
- [ ] Define a function named `show_welcome_message`.
- [ ] Inside the function, print a welcome message for the library.
- [ ] Call `show_welcome_message`.
- [ ] Define a function named `greet_reader`.
- [ ] Give `greet_reader` one parameter named `name`.
- [ ] Inside the function, print a greeting using `name`.
- [ ] Call `greet_reader` with your name as the argument.
- [ ] Call `greet_reader` again with a different name as the argument.
- [ ] Create a list named `books` with at least three book titles.
- [ ] Define a function named `display_books`.
- [ ] Give `display_books` one parameter named `book_list`.
- [ ] Inside the function, use a `for` loop to print each book.
- [ ] Call `display_books` and pass in the `books` list.
- [ ] Add a comment explaining what a function is.
- [ ] Add a comment explaining what a parameter is.
- [ ] Add a comment explaining what an argument is.
- [ ] Add a comment explaining why the function body does not run until the function is called.
