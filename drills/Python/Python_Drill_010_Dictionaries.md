# Python Drill - Dictionaries

## Objective

Practice using dictionaries to group related data and use a list of dictionaries to manage multiple books.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Create a dictionary named `book`.
- [ ] Give the dictionary these keys:
  - `title`
  - `author`
  - `checked_out`
- [ ] Store `"Dune"` as the title.
- [ ] Store `"Frank Herbert"` as the author.
- [ ] Store `False` as the checked out status.
- [ ] Print the full `book` dictionary.
- [ ] Print only the book title using the `title` key.
- [ ] Print only the author using the `author` key.
- [ ] Change the `checked_out` value to `True`.
- [ ] Print the updated `checked_out` value.
- [ ] Add a new key named `year`.
- [ ] Store `1965` as the year.
- [ ] Print the full updated dictionary.
- [ ] Create a list named `books`.
- [ ] Add at least three dictionaries inside the `books` list.
- [ ] Each dictionary should represent one book.
- [ ] Each book dictionary should have:
  - `title`
  - `author`
  - `checked_out`
- [ ] Use a `for` loop to print each book title.
- [ ] Use a `for` loop to print each book title and author.
- [ ] Use a `for` loop and an `if` statement to print whether each book is available or checked out.
- [ ] Manually change one book’s `checked_out` value to `True`.
- [ ] Print the updated library status.
- [ ] Add a comment explaining what a dictionary is.
- [ ] Add a comment explaining what a key-value pair is.
- [ ] Add a comment explaining what `book["title"]` does.
- [ ] Add a comment explaining what `books[1]["checked_out"] = True` does.
