# Python Drill 006 - Comparison Operators

## Objective

Practice using comparison operators to make true or false checks.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Create a variable named `checkout_limit` and set it to `3`.
- [ ] Ask the user how many books they want to check out.
- [ ] Convert the response to an integer.
- [ ] If the requested number is greater than the checkout limit, print an over-limit message.
- [ ] If the requested number is less than or equal to the checkout limit, print an approved message.
- [ ] Ask the user for their name.
- [ ] Clean and format the name.
- [ ] Check whether the name is not empty.
- [ ] If the name is not empty, greet the user.
- [ ] If the name is empty, print a message saying the name is required.
- [ ] Add a comment explaining the difference between `=` and `==`.
