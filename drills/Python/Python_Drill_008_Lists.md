# Lists

## Objective

Practice creating, accessing, adding to, removing from, and counting items in a list.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Create a list named `books` with three book titles.
- [ ] Print the whole list.
- [ ] Print the first book using its index.
- [ ] Print the second book using its index.
- [ ] Print the third book using its index.
- [ ] Add one new book to the list using `.append()`.
- [ ] Print the updated list.
- [ ] Count the number of books using `len()`.
- [ ] Store the count in a variable named `book_count`.
- [ ] Print the book count.
- [ ] Remove one book from the list using `.remove()`.
- [ ] Print the updated list.
- [ ] Add a comment explaining why list indexes start at `0`.
- [ ] Add a comment explaining the difference between `IndexError` and `ValueError`.
