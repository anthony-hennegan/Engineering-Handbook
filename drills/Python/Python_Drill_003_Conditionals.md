# Python Drill - Conditionals

## Objective

Practice using `if`, `elif`, and `else` to make a program choose between different paths.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Ask the user if the library is open.
- [ ] If the user types `yes`, print a message saying the library is open.
- [ ] If the user types `no`, print a message saying the library is closed.
- [ ] If the user types anything else, print a message asking them to type yes or no.
- [ ] Ask the user if they want to view the book selection.
- [ ] If the user types `yes`, print a message saying the books will be shown.
- [ ] If the user types `no`, print a polite goodbye message.
- [ ] If the user types anything else, print a message asking them to type yes or no.
- [ ] Use at least one boolean variable in an `if` statement.
