# Python Drill 005 - Numbers and Type Conversion

## Objective

Practice converting user input from strings into numbers.

---

## Rules

- No handbook
- No flashcards
- No internet
- Run the program from the terminal
- Write the code from memory

---

## Tasks

- [ ] Create a new Python file for the drill.
- [ ] Ask the user how many books they want to check out.
- [ ] Convert the response to an integer.
- [ ] If the number is greater than 3, print a message saying they can only check out 3 books.
- [ ] Otherwise, print a message saying the checkout amount is approved.
- [ ] Ask the user for a late fee amount.
- [ ] Convert the response to a float.
- [ ] Print the late fee amount.
- [ ] Create a number variable.
- [ ] Convert that number to a string.
- [ ] Print the converted string.
- [ ] Print the data type of each converted value.
