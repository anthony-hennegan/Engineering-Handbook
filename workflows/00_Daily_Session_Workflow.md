```markdown
# Coding Dojo Session Workflow

## Phase 1 - Warm Up

- [ ] Complete all due Anki reviews.
- [ ] Review the previous drill (if needed).
- [ ] Open the current project and handbook.

---

## Phase 2 - Learn

- [ ] Introduce today's topic.
- [ ] Explain the concept in your own words.
- [ ] Refine the explanation together.
- [ ] Complete the handbook notes.

---

## Phase 3 - Build

- [ ] Apply the new concept to the project.
- [ ] Test the code frequently.
- [ ] Ask questions as they arise.

---

## Phase 4 - Reinforce

- [ ] Create flashcards.
- [ ] Complete the drill without notes.
- [ ] Update the handbook with anything learned during the drill.

---

## Phase 5 - Close Session

- [ ] Commit one logical unit of work.
- [ ] Push to GitHub.
- [ ] Decide on the objective for the next session.
```
